# 🌐 Personal Portfolio Website

A modern and responsive **personal portfolio website** built to showcase my skills, projects, resume, and contact information.  
This project is designed to represent my work as a web developer in a clean and professional way.

🔗 Live Website: https://demo-portfolio-three-rho.vercel.app/

---

## 📌 About the Project

This portfolio website is created to highlight my:
- Skills
- Projects
- Resume
- Social profiles (GitHub, LinkedIn, etc.)
- Contact details

It helps in building my personal brand and showcasing my development journey.

---

## ✨ Features

- Clean and modern UI design  
- Fully responsive (Mobile, Tablet, Desktop)  
- Smooth scrolling navigation  
- Hero section with introduction  
- About section  
- Projects showcase section  
- Resume download/view section  
- Contact section  
- Social media links (LinkedIn, Instagram, Email, etc.)
